# AI негізіндегі ауыл шаруашылық мониторинг жүйесі

## Жоба сипаттамасы
Бұл жоба спутниктік деректерді пайдаланып егістік алқаптарының жағдайын талдауға арналған. Жүйе NDVI индексін есептеп, уақыт бойынша өзгерістерді анықтайды және жасанды интеллект көмегімен проблемалық аймақтарды табады. AI әдісі ретінде Isolation Forest алгоритмі қолданылды.

## Мақсаты
Фермерлерге нақты шешім қабылдауға көмектесу және өнім шығынын азайту.

## Мүмкіндіктері
- NDVI есептеу
- Өзгерістерді анықтау
- AI анализ
- Қауіпті аймақтарды анықтау

## Қолданылған технологиялар
- Python
- NumPy
- Matplotlib

## Нәтиже
